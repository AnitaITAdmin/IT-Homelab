# Active Directory

## Ziel

Aufbau einer Active-Directory-Domäne für eine Windows-Testumgebung.

## Testumgebung

- Windows Server 2022
- Windows 11
- VirtualBox

## Aufgabe

Installation und Konfiguration der Active Directory Domain Services (AD DS) sowie Einrichtung einer neuen Active-Directory-Domäne.

## Durchführung

- Installation der Rolle Active Directory Domain Services (AD DS)
- Heraufstufung des Servers zum Domänencontroller
- Erstellung einer neuen Gesamtstruktur (Forest)
- Konfiguration der DNS-Integration
- Überprüfung der erfolgreichen Domänenerstellung

## Ergebnis

Die Active-Directory-Domäne wurde erfolgreich eingerichtet und bildet die Grundlage für die weitere Konfiguration der Testumgebung.

## Erkenntnisse

Active Directory bildet die Grundlage einer Windows-Domäne und ermöglicht die zentrale Verwaltung von Benutzern, Computern, Gruppen und Sicherheitsrichtlinien.
