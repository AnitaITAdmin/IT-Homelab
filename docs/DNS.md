# DNS

## Ziel

Konfiguration eines DNS-Servers für die Namensauflösung innerhalb einer Active-Directory-Domäne.

## Testumgebung

- Windows Server 2022
- Windows 11
- VirtualBox

## Aufgabe

Installation und Konfiguration des DNS-Servers als Bestandteil der Active-Directory-Domäne.

## Durchführung

- Überprüfung der DNS-Serverkonfiguration
- Kontrolle der Forward-Lookup-Zone
- Erstellung und Überprüfung von DNS-Einträgen
- Namensauflösung mit nslookup getestet
- Erreichbarkeit mit ping überprüft

## Ergebnis

Die Namensauflösung innerhalb der Domäne funktioniert zuverlässig. Clients und Server können sich gegenseitig über DNS auflösen.

## Erkenntnisse

DNS ist ein zentraler Bestandteil einer Active-Directory-Umgebung und ermöglicht die zuverlässige Kommunikation zwischen Domänencontrollern, Servern und Clients.
