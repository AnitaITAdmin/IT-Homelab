# Netzwerk

## Ziel

Konfiguration und Überprüfung der grundlegenden Netzwerkkommunikation innerhalb einer Windows-Domänenumgebung.

## Testumgebung

- Windows Server 2022
- Windows 11
- VirtualBox

## Verwendete Werkzeuge

- Eingabeaufforderung (CMD)
- ipconfig
- ping
- nslookup

## Aufgabe

Überprüfung der Netzwerkkonfiguration sowie der Kommunikation zwischen Client und Server.

## Durchführung

- IP-Konfiguration überprüft
- Netzwerkerreichbarkeit mit ping getestet
- DNS-Namensauflösung mit nslookup überprüft
- Netzwerkeinstellungen mit ipconfig kontrolliert
- Verbindung zum Domänencontroller getestet

## Ergebnis

Die Netzwerkkommunikation zwischen Client und Server funktioniert zuverlässig. Die Namensauflösung und die Erreichbarkeit aller Systeme wurden erfolgreich überprüft.

## Erkenntnisse

Eine korrekte IP-Konfiguration sowie eine funktionierende DNS-Namensauflösung sind die Grundlage für den Betrieb einer Active-Directory-Umgebung.
