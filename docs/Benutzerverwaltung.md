# Benutzerverwaltung

## Ziel

Verwaltung von Benutzerkonten und Sicherheitsgruppen innerhalb einer Active-Directory-Domäne.

## Testumgebung

- Windows Server 2022
- Windows 11
- VirtualBox

## Verwendete Werkzeuge

- Active Directory-Benutzer und -Computer (ADUC)
- Server-Manager

## Aufgabe

Erstellung und Verwaltung von Benutzerkonten sowie Organisation der Zugriffsberechtigungen über Sicherheitsgruppen.

## Durchführung

- Benutzerkonten erstellt
- Organisationseinheiten (OU) eingerichtet
- Sicherheitsgruppen erstellt
- Benutzer den entsprechenden Gruppen zugewiesen
- Gruppenmitgliedschaften überprüft

## Ergebnis

Die Benutzerverwaltung wurde erfolgreich eingerichtet. Zugriffsrechte können zentral über Sicherheitsgruppen verwaltet werden.

## Erkenntnisse

Eine strukturierte Benutzer- und Gruppenverwaltung vereinfacht die Administration und ermöglicht eine sichere Vergabe von Berechtigungen.
