# PowerShell

## Ziel

Nutzung von PowerShell zur Überprüfung und Verwaltung grundlegender Windows-Systemfunktionen.

## Testumgebung

- Windows Server 2022
- Windows 11
- PowerShell

## Verwendete Werkzeuge

- Windows PowerShell
- PowerShell ISE

## Aufgabe

Ausführung grundlegender administrativer Befehle zur System- und Netzwerkdiagnose.

## Durchführung

- Prozesse mit Get-Process überprüft
- Dienste mit Get-Service kontrolliert
- Netzwerkverbindung mit Test-Connection getestet
- IP-Konfiguration mit Get-NetIPConfiguration überprüft
- Active-Directory-Benutzer mit Get-ADUser abgefragt
- Ereignisprotokolle mit `Get-W
