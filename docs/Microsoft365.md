# Microsoft 365

## Ziel

Einrichtung und Verwaltung einer Microsoft-365-Umgebung für eine Unternehmensinfrastruktur.

## Testumgebung

- Microsoft 365 Business Standard
- Microsoft Entra ID
- Exchange Online
- Microsoft Teams

## Verwendete Werkzeuge

- Microsoft 365 Admin Center
- Microsoft Entra Admin Center
- Exchange Admin Center
- Microsoft Teams Admin Center

## Aufgabe

Einrichtung einer Microsoft-365-Umgebung einschließlich Benutzerverwaltung, Gruppen, Exchange Online und Microsoft Teams.

## Durchführung

### Benutzerverwaltung

- Benutzerkonten erstellt
- Lizenzen zugewiesen
- Sicherheitsgruppen verwaltet

### Microsoft Entra ID

- Benutzer und Gruppen organisiert
- Rollen zugewiesen
- Geräteübersicht geprüft

### Exchange Online

- Postfächer verwaltet
- Shared Mailbox erstellt
- E-Mail-Einstellungen konfiguriert

### Microsoft Teams

- Teams und Kanäle erstellt
- Benutzer hinzugefügt
- Zugriff auf Teams und Kanäle überprüft

## Ergebnis

Die Microsoft-365-Umgebung wurde erfolgreich eingerichtet. Benutzer, Gruppen und Dienste arbeiten gemeinsam innerhalb einer zentral verwalteten Cloud-Infrastruktur.

## Erkenntnisse

Microsoft 365 ermöglicht die zentrale Verwaltung von Benutzern, Kommunikation und Zusammenarbeit innerhalb eines Unternehmens.
