# Dateiserver & NTFS-Berechtigungen

## Ziel

Einrichtung eines Dateiservers sowie Konfiguration von NTFS-Berechtigungen für eine sichere Datenverwaltung.

## Testumgebung

- Windows Server 2022
- Windows 11
- VirtualBox

## Verwendete Werkzeuge

- Server-Manager
- Datei-Explorer
- Active Directory-Benutzer und -Computer (ADUC)

## Aufgabe

Freigabe eines Ordners im Netzwerk sowie Konfiguration von Freigabe- und NTFS-Berechtigungen für Benutzer und Gruppen.

## Durchführung

- Freigegebenen Ordner erstellt
- Netzwerkfreigabe konfiguriert
- NTFS-Berechtigungen eingerichtet
- Sicherheitsgruppen für den Zugriff verwendet
- Zugriff mit verschiedenen Benutzerkonten getestet

## Ergebnis

Der Zugriff auf den freigegebenen Ordner funktioniert entsprechend den konfigurierten Berechtigungen. Benutzer erhalten ausschließlich die vorgesehenen Zugriffsrechte.

## Erkenntnisse

Durch die Kombination von Freigabe- und NTFS-Berechtigungen lassen sich Zugriffe auf Unternehmensdaten zentral und sicher verwalten.
